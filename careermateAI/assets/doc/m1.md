## 一、現有代碼需要修改的地方

### 1.1 styles.css — `.footer-links` 繼承了 `.nav-links` 的絕對定位

**問題：** `.footer-links` 和 `.nav-links` 共用同一條 CSS 規則，其中 `position: absolute; left: 50%; transform: translateX(-50%)` 是 navbar 的居中技巧，放在 flex 佈局的 footer 裡會導致位置錯亂。

**修改（styles.css）：**
```css
/* 修改前（現在） */
.footer-links,
.nav-links {
    display: flex;
    gap: 40px;
    position: absolute;
    left: 50%;
    transform: translateX(-50%);
}

/* 修改後：拆開兩條規則 */
.nav-links {
    display: flex;
    gap: 40px;
    position: absolute;
    left: 50%;
    transform: translateX(-50%);
}

.footer-links {
    display: flex;
    gap: 32px;
}
```

---

### 1.2 styles.css — `.btn-hero-primary` 的 transition 不完整

**問題：** transition 只宣告了 `background-color`，但 hover 改變的是 `box-shadow` 和 `transform`，導致這兩個屬性在 hover 時突變（無動畫）。

**修改（styles.css）：**
```css
/* 修改前 */
.btn-hero-primary,
.btn-hero-secondary {
    transition: background-color 0.3s ease;
}

/* 修改後 */
.btn-hero-primary,
.btn-hero-secondary {
    transition: background-color 0.3s ease, box-shadow 0.3s ease, transform 0.3s ease;
}
```
