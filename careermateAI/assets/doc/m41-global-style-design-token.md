Summary: 使用 CSS 变量配置全局样式基础，统一项目视觉风格与排版系统。

Description:

在 CSS 中设置全局变量（Design Tokens）：

颜色（主色、背景色、文字色等）

圆角（border-radius）

阴影（box-shadow）

间距（padding, margin, spacing scale）

最大宽度（container 宽度）

设置基础样式：

字体与字号系统（如标题、正文等）

背景色、默认链接样式

Acceptance Criteria:

所有基础样式通过 CSS 变量集中管理。

页面整体视觉统一，无明显字号/间距混乱。

链接样式、基础排版已统一设置。
